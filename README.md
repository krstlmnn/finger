# Finger

A finger client for the inferno operating system, written in limbo.

```
FINGER(1)                                               FINGER(1)

NAME
     finger - query a remote user information program

SYNOPSIS
     finger [ -v ] user@host

DESCRIPTION
     Finger queries host for information about user and copies
     the response to standard output.

     The -v option causes finger to request for more verbose
     output.

SOURCE
     https://github.com/krstlmnn/finger

SEE ALSO
     ``The Finger User Information Protocol'', RFC1288.

                         Inferno Manual
```
